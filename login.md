# Módulo de Login - PetroSafe

## Descripción
Este módulo es responsable de la autenticación de usuarios en la plataforma PetroSafe.

## Objetivo
Garantizar el acceso seguro al sistema mediante la validación de credenciales.

## Estructura
- `src/controllers/`: Contiene la lógica del Login.
- `docs/login.md`: Documentación técnica.

## Instalación y Ejecución
Para probar este módulo, compila el proyecto en Android Studio y ejecuta la actividad LoginActivity.
